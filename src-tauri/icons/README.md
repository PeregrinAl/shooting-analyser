# Плейсхолдер-иконки

Файлы `32x32.png`, `128x128.png`, `128x128@2x.png` — это **временные
заглушки**: однотонные тёмно-серые квадраты, сгенерированные через stdlib
Python. Они нужны только потому, что макрос `tauri::generate_context!`
читает иконки во время компиляции, иначе ни `cargo check`, ни `npm run
tauri:dev` не пройдут.

Перед первым релизом замените их настоящим логотипом командой

```sh
npm run tauri icon path/to/logo.png
```

Эта команда сгенерирует полный набор размеров, включая `icon.ico` (Windows)
и `icon.icns` (macOS), которые нужны установщикам. После этого добавьте
.icns/.ico в `bundle.icon` в `src-tauri/tauri.conf.json`.

Напоминание: в логотипе и иконке **не использовать слово SCATT** и иные
товарные знаки производителя — см. ТЗ.
